# webui.string.clipboard( text )
> 内容复制到剪切板

## 参数

| 属性 | 类型   | 必填 | 说明           |
| ---- | ------ | ---- | -------------- |
| text | string | 是   | 需要复制的内容 |

## 示例代码

```javascript
webui.string.clipboard('WEBUI 前端框架');
```
