webui.loading( align )

Bootstrap loading 加载动画

参数

属性 类型 默认值 说明
align string center 动画位置

string.center 的合法值

值 说明
start 左对齐
end 右对齐
center 居中

示例代码

左对齐

webui.loading('start');

右对齐

webui.loading('end');

居中

webui.loading('center');
